# Raw — real-world data

The unprocessed flow of real-world data: user reports, logs, research, market
signals, transcripts — whatever arrives, as it arrived.

An item leaves this folder when the PRD absorbs it — which supersedes the PRD,
and the retired version records the dated folder that caused it.

## Structure

One folder per arrival date, `YYYY-MM-DD`:

```
raw/2026-07-16/
raw/2026-08-02/
```

## What goes here
- Real-world data in whatever form it showed up
- Nothing derived, nothing cleaned, nothing summarised

Capture rules in [`AGENTS.md`](AGENTS.md).
