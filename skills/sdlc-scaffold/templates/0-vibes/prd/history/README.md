# PRD history

Retired versions of the PRD, one file per superseded version.

Nothing leaves this folder. It is forensic: it answers *why the PRD changed*,
never *what the PRD says*. The current PRD is the only answer to that, and it
keeps its deprecated requirements so no one needs to look here to resolve a
citation.

Each file is named for the date it was superseded (`PRD-2026-07-16.md`) and
opens with a header naming when it was retired, why, and which `raw/` folder
caused it.
