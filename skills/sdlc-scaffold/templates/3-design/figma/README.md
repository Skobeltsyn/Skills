# Figma — design source of truth

The design origin for stage 3. Every component in `../react/` and `../vue/` is
derived from, or synced against, the Figma file linked in
[`AGENTS.md`](AGENTS.md).

A component leaves this folder when it exists in at least one framework folder
with a Storybook story covering all of its internal states.

## What goes here

Only the pointer to the Figma file and the conventions governing it — the design
artifacts themselves live in Figma, not in the repo. The file URL is a `TODO`
until the adaptation pass sets it.

Naming and sync rules in [`AGENTS.md`](AGENTS.md).
