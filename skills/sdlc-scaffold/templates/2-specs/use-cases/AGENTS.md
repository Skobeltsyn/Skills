USE-CASES naming: UC-{number}-ACTOR-{number}-EVT-{number}-ENT-{number}-RESULT-IN-MODULE

The filename carries the traceability: primary actor, trigger event, primary
entity, outcome. A directory listing alone should tell you who did what to which
entity, and how it ended.

Every id cited in the name must already exist in ../actors/, ../events/, and
../entities/ — a use-case cannot invent its own.
