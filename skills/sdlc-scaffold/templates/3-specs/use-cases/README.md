# Use-cases

The **how it plays out** — end-to-end scenarios that tie an actor, a trigger
event, and a primary entity to a result.

Per `../CLAUDE.md`, each use-case is a separate file named
`UC-{number}-ACTOR-{number}-EVT-{number}-ENT-{number}-RESULT`, encoding the
primary actor, trigger event, primary entity, and outcome. The filename carries
the traceability — a directory listing alone should tell you who did what to
which entity, and how it ended.
